## Opdracht

Nu ben jij de computer. Werk het algoritme uit. Geef a.d.h.v. 3 regels de eindwaarden van `a`, `b` en `c` weer. Gebruik hiervoor de `print()` functie


## Algoritme

```
a = 1
b = 2
a = a+b
c = b+a
b += a
a = a+c-a
```

*output*
```
print(<waarde a>)
print(<waarde b>)
print(<waarde c>)
```
